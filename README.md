# Ariana Gamardo - Rumbo al Mundial de Taekwondo España 2026

Página web de aterrizaje (Landing Page) interactiva y premium para la captación de patrocinadores individuales y corporativos para **Ariana Gamardo**, atleta juvenil argentina de Taekwondo ITF que compite en el Campeonato Mundial de España.

## 🚀 Características
*   **Diseño Premium**: Modo oscuro moderno con acentos en celeste neón y dorado de alta fidelidad. Responsive y adaptado para dispositivos móviles y tablets.
*   **Simulador de Aportes**: Panel financiero interactivo donde los interesados pueden simular contribuciones y previsualizar qué gastos específicos (inscripción, equipamiento, estadía, pasajes) se estarían cubriendo.
*   **Herramienta de Copiado Rápido**: Botones interactivos con tooltips para copiar el CBU y Alias de Mercado Pago.
*   **Esquema de Auspicio Corporativo**: Tabla de patrocinio detallada para marcas con enlaces directos pre-rellenados en el formulario de contacto.
*   **Formulario de Contacto**: Formulario validado en tiempo real con animación de carga e indicación de envío exitoso.
*   **Optimización SEO**: Etiquetas meta estructuradas, Open Graph para redes sociales, y carga asíncrona/lazy con Intersection Observer.

## 🛠️ Tecnologías
*   HTML5 Semántico
*   CSS3 Vanilla (Gradients, Variables, Flexbox, Grid, Glassmorphism, CSS Animations)
*   JavaScript Vanilla (Intersection Observer, DOM, Event Listeners, Clipboard API)
*   [http-server](https://www.npmjs.com/package/http-server) para desarrollo local

## 💻 Instalación y Uso Local

1. Cloná este repositorio o descargá los archivos.
2. Asegurate de tener [Node.js](https://nodejs.org/) instalado en tu sistema.
3. Abrí tu terminal en el directorio del proyecto y ejecutá:
   ```bash
   npm run dev
   ```
4. El sitio web se abrirá automáticamente en tu navegador bajo el puerto `3000` (`http://localhost:3000`).
